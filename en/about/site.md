# Site Introduction

I started building this site because tutorial quality online is inconsistent — especially on domestic forums like CSDN, Juejin, and Jianshu. So I decided to create my own — detailed, beginner-friendly tutorials that not only push me to learn, but also share better resources to save everyone's time.

One person can't cover everything, so contributions are welcome to build a high-quality knowledge base together.

## Logo Origin

The site logo comes from a Google emoji combiner I found on a post — Clown + Angry.

## Metadata & Conventions

| Element | Description |
|---------|-------------|
| Sidebar | Links to topics within the current section, or tutorial sequence |
| Section landing page | Introduction to the section's scope and purpose |
| Search | Top-left search bar, or `Ctrl+K` to open quickly |
| Comments | Available at the bottom of each page |
