Vue.js 是一个渐进式 JavaScript 框架，用于构建用户界面。易学易用，性能出色。

## 核心特性

- **响应式数据绑定**：数据变化自动更新视图
- **组件化开发**：可复用的组件系统
- **虚拟 DOM**：高效的 DOM 更新
- **Composition API**：Vue 3 推荐的逻辑组织方式

## 快速开始

```bash
npm create vue@latest
cd my-app
npm install
npm run dev
```
