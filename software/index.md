Desktop clients are the software users run — `.exe`, `.apk`, `.ipa` files. Desktop development is the process of building these applications.

## Tauri

A Rust-based cross-platform app framework. Write once, build for multiple platforms. Already has mature products like [ChatWise](https://chatwise.app) and [HuLa](https://hulaspark.com/).
