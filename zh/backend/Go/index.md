# 为什么学 Go

我学习的第一个编程语言是 Python，不得不承认 Python 确实非常便捷、强大，但性能不够强劲。换个方向思考，一些使用频率很高的工具还是用运行效率高的语言写好一点，而且还要考虑到并发需求，最终选择学一门高性能语言。

Go 作为 Python 友好学习的语言，最终被我选中。

> 学习路线仅供参考：

1. [Go 语言之旅](https://tour.go-zh.org/welcome/1) + [Bilibili 视频](https://www.bilibili.com/video/BV1ae41157o9) 掌握基本语法
2. GoFrame 初试（后来发现过于全面而放弃）
3. Gin + Gorm 搭建后端
4. 待续……
