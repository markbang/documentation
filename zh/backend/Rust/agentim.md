# AgentIM — Rust 多平台 IM 桥接

AgentIM 是一个 Rust 桥接，将聊天平台连接到本地 AI 代理后端（Codex）。通过 webhooks 和长轮询支持 8+ 平台，具备会话持久化、上下文裁剪和路由规则。
