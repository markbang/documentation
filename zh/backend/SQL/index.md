MySQL 是最流行的开源关系型数据库管理系统，广泛应用于 Web 应用开发。

## 核心特性

- **高性能**：优化的查询引擎
- **可靠性**：ACID 事务支持
- **可扩展性**：支持主从复制、分库分表
- **开源免费**：社区版完全免费
