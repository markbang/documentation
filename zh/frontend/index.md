## 核心技术栈

| 技术 | 作用 | 说明 |
|------|------|------|
| HTML | 页面结构 | 定义网页骨架 |
| CSS | 样式与布局 | 颜色、字体、间距 |
| JavaScript | 交互逻辑 | 动态功能与事件处理 |

## 学习路径

1. **基础阶段**：HTML / CSS / JavaScript + DOM 与事件 + ES6+ 新特性
2. **进阶阶段**：前端框架（Vue / React）+ TypeScript + 构建工具
3. **高级阶段**：性能优化 + 工程化实践 + 跨端开发
